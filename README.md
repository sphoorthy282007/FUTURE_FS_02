# CRM Lead Management System

A full-stack CRM application built using:

## Frontend
- React.js
- Vite
- Framer Motion
- Recharts

## Backend
- Node.js
- Express.js
- MySQL
- JWT Authentication

## Features
- User Login
- Lead Management
- Dashboard Analytics
- Lead Conversion Tracking
- Notes Management

## Run Project

Backend:

cd backend
npm install
npm run dev

Frontend:

cd frontend
npm install
npm run dev
